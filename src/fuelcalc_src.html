ABOUT THIS FUEL PLANNING TOOL

Disclamer / Privacy policy

This tool is provided to only make your fuel planning for playing with a Flight Simulator. Do not use this tool for real life flight planning!

This tool works completely offline and standalone except you want to obtain current metar data. So it comes without any privacy policy. If you load the current METAR data the requested information will be loaded from https://www.checkwx.com. Therefore a connection is established to that server. Please read their privacy policy to be informed how they are dealing with your personal information.

Get the tool and all related files at GitHub and checkout the manual there. If you have any questions, issues or improvements please let me know at GitHub. Feel free to fork and improve this tool:

https://github.com/el-mejor/Fuelplanner
